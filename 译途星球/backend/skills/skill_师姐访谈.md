# 技能:师姐访谈

使用 `get_story` 工具,获取一个 AI 生成的"师姐访谈"故事。

## 设计理念
真实的往届案例比任何数据都有力量。这些故事是 AI 基于典型英专生职业路径生成的,
每个故事都有一个核心信息:**"你也可以。"**

## 输出格式
```json
{
  "name": "师姐化名",
  "background": "原专业/背景",
  "current_role": "现在在做什么",
  "quote": "一句有力量的原话",
  "advice": "给学弟学妹的建议"
}
```

## 故事必须传递的信号
- "她和你一样迷茫过"——打破孤独感
- "她做的不一定是最'厉害'的工作,但是她自己选的"——拓宽"好工作"的定义
- "她的一个具体动作改变了轨迹"——给用户一个可模仿的行为
