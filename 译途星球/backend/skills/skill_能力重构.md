# 技能:能力重构

使用 `tag_reconstruct` 工具,把用户选的"不起眼的小事"翻译成"职场竞争力"。

## 核心理念
英专学生最大的困境不是"没有能力",而是**不会翻译自己的能力**。
"我帮字幕组翻了 200 集美剧" = 项目管理 + 跨文化沟通 + 细节把控 + 抗压能力
"我给爱豆做应援" = 社群运营 + 内容创作 + 活动策划 + 粉丝经济理解

## 标签分类(用户看到的)
- 📺 内容与故事:字幕组翻译、写同人文/追星文案、做 B 站视频/小红书笔记
- 🌍 语言与沟通:帮同学改简历/作文、做展会翻译志愿者、教留学生中文
- 📊 分析与细节:校对/排版、整理资料做表格、写读书笔记/影评
- 🤝 组织与协作:组织社团活动、做小组作业组长、策划班级聚会
- 💻 数字与工具:做公众号/排版、剪视频、做 PPT/海报

## AI 重构逻辑
每个标签翻译为:
- **职场能力名称**:把这个标签翻译成 HR 看得懂的能力
- **一句话描述**:描述这个能力为什么重要
- **匹配方向**:这个能力可以做什么

## 重构示例
| 用户标签 | 翻译成 | 匹配方向 |
|---|---|---|
| 字幕组翻译 | 项目管理 + 跨文化沟通 + 抗压能力 | 游戏本地化、内容运营、海外市场 |
| 追星应援/写文案 | 内容创作 + 社群运营 + 粉丝经济 | 新媒体运营、品牌营销、社区管理 |
| 帮同学改简历 | 编辑审校 + 共情沟通 + 知识迁移 | 教育培训、内容编辑、人力资源 |
| 做小组组长 | 团队协作 + 项目管理 + 冲突解决 | 项目管理、客户成功、销售管理 |
| 剪视频/做公众号 | 内容制作 + 视觉设计 + 数据分析 | 视频制作、数字营销、UI/UX 设计 |
