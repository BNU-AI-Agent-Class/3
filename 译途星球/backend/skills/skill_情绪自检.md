# 技能:情绪自检(GAD-7)

使用 `gad7_score` 工具,帮用户完成 GAD-7 焦虑自评量表。

## 使用时机
- 用户选择了"陪我聊聊现在的心情"
- 用户主动提到"我最近很焦虑""我压力很大"

## 量表说明
- 7 道题,每题 0-3 分,总分 0-21
- 仅作自我了解参考,绝不做任何诊断
- 不强制用户完成——随时可以跳过

## 分数反馈话术

| 分数 | 等级 | 话术要点 |
|---|---|---|
| 0-4 | 轻度 | "你的焦虑水平在正常范围内"——肯定她的韧性,邀请进入能力重构 |
| 5-9 | 中度 | "大四面临选择,谁都会紧张"——正常化她的感受,放缓节奏 |
| 10-14 | 中重度 | "我感觉到你最近压力很大"——先陪伴,不急着推进职业内容 |
| 15-21 | 重度 | "你最近一定很累吧"——建议休息,不推进任何内容 |

## 重要边界
- **GAD-7 分数不触发危机协议。**危机协议只由用户主动表达的自伤/自杀信号触发。
- 所有话术中绝不说"你有焦虑症""你抑郁了"。
- 结果后必须加一句:"这只是让你了解自己状态的小工具,不是诊断,不是标签。"
